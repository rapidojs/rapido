Rapido

Build universal native apps incredibly fast.

Get Started

Fast Development

Rapido makes it fast to spin up an app and start developing features right away. No tedious boilerplate work. Just run a single command and start developing features immediately.

Powerful Tools

The Rapido toolset uses Expo, Styled Components, React Navigation and other amazing projects under the hood to power your app. Rapido tools are broken up into multiple packages to serve different common needs such as navigation, device storage, component library, etc.

No Lock-In

All Rapido tools are broken up into multiple packages and are all optional. You can use as much or as little of the Rapido toolset to serve your needs. The core scripts package is ejectable if advanced configuration is desired.

Get Started In Seconds

To create a new Rapido project called my-app, run this command:

npx @rapido/init my-app

Easy To Maintain

When new versions of Rapido tools are released, you can upgrade using a single command:

yarn run upgrade
