# Rapido Website

This is the official [Rapido](https://rapidojs.org) website code. It is built using Docusaurus 2, a modern static website generator.

Refer to the official Docusaurus [documentation](https://v2.docusaurus.io) before making significant changes to the website.

### Installation

```
$ yarn
```

### Local Development

```
$ yarn start
```

This command starts a local development server and open up a browser window. Most changes are reflected live without having to restart the server.
